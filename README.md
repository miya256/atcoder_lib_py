# 競技プログラミングの自作ライブラリ
コードの一番上にコピペして使いましょう

## フォルダ
### algorithm/  
  旧ライブラリ。libraryに移植中。
### library/  
  新しいライブラリ。ファイルが増えてごちゃついてきたり、型ヒントやdocstringなどをちゃんとつけたくなったため。
### test/  
  コンテスト中にコードを書くファイルなど。自動テストもある。

## 自動テスト
### 機能
- 開いているページの問題文や入出力例をスクレイピング
- すべての入力例で自分のコードを実行し、出力例と一致するか自動で判定
- 昇順で出力することやmodを取ることなど、見逃しやすい項目が問題文に含まれていれば、警告する
- assert文を消すなど、コードを整えて、クリップボードにコピー
